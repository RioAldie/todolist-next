# Todolist

## stack and tools

    - next
    - next-auth
    - mongoo db
    - mongoose
    - sonner
    - eslint
    - mongoodb atlas

## run in local

First, download the zip file or clone this project and install the node_modules

```
npm install
```

run the development server:

```
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
